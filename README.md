ECC
===

A simple Elliptic Curve Calculator

## Prerequires

- [Ruby](https://www.ruby-lang.org/en/)
- [Sinatra](http://www.sinatrarb.com/)

## Installing

1. Installing Ruby:

        $ sudo apt-get install ruby

2. Install Sinatra

        $ sudo gem install sinatra

3. Run the website

        $ ruby web.rb
